Evergreen Classifier API

Author: Oweys Momenzada

Date of deployment: 15.09.2021

 

Attribute:

  • {"label" : "Evergreen" , "Ephemeral", "Uncertain" }  - Input enthält überwiegend Evergreen-, Ephemeralcontent oder der Classifier ist sich nicht sicher (Uncertain).
  • {"evergreen_probability": Float} - Wahrscheinlichkeit (in %), dass der Input Evergreen ist.
  • {"is_truncated": Bool} - Wurde der Text gekürzt? Maximal Länge eines Textes sind 512 Wörter.

 

Code:

import requests

url = "secret.url.app/api"

def evergreen_clf_pred(text):
    json_text = {'text' : text}
    pred = requests.post(url,json=json_text)

    return pred.json()

 

Beispiel:

example = evergreen_clf_pred("Nun der Wetterbericht von heute")

print(example)
