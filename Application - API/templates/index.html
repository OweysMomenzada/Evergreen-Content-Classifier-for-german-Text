Evergreen Classifier API

Author: Oweys Momenzada

Date of deployment: 14.09.2021

 

Attribute:

  • {"label" : "Evergreen" , "Ephemeral", "Uncertain" }  - Input enthält überwiegend Evergreen-, Ephemeralcontent oder der Classifier ist sich nicht sicher (Uncertain).
  • {"evergreen_probability": Float} - Wahrscheinlichkeit (in %), dass der Input Evergreen ist.
  • {"is_truncated": Bool} - Wurde der Text gekürzt? Maximal Länge eines Textes sind 512 Wörter.

 

Code zum kopieren:

import urllib, json
import urllib.request as urllib2
import codecs

 

def make_url_clean(url):
    word = codecs.encode(url,'utf-8')
    url_Word = urllib2.quote(url)

    return url_Word

 

def predict_evergreen(text):
    url_clean = make_url_clean(text)
    url = "http://127.0.0.1:9000/" + url_clean
    response = urllib.request.urlopen(url)
    data = json.loads(response.read())

    return data

 

results = predict_evergreen("Es hagelt!")

print(results)
